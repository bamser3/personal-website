# Personal Website

This repository contains the source code for my personal website. It serves as a portfolio to showcase my projects, skills, and background as a software developer.

---

## Features
- About Me section introducing my background and interests
- Skills section summarizing technical abilities
- Projects section highlighting selected work with descriptions and images
- Interactive project section including hover effects and transitions

---

## Tech Stack
- HTML and CSS for structure and styling
- Flexbox and CSS Grid for responsive layouts
  
---

## Setup and Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/personal-website.git
    ```
2.cd into the directory:
  ```bash
  cd personal-website
  ```

4. open the index.html file in your browser to view the site.
